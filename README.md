<img width="440" height="193" alt="image" src="https://github.com/user-attachments/assets/8a8dd3d0-87d6-47f5-bec8-7d749cfcd70b" />#  Deadline Tracker Bot

Telegram-бот для управления дедлайнами с умными напоминаниями и повторяющимися задачами.

Помогает студентам и всем, кто работает с дедлайнами, не забывать о важных датах.

##  Возможности

### Основные
-  Добавление дедлайнов через интерактивный календарь
-  Просмотр всех дедлайнов с сортировкой по дате
-  Автоматический подсчёт оставшихся дней
-  Редактирование названий и дат дедлайнов
-  Удаление дедлайнов
-  Постоянное хранение данных (SQLite)

### Продвинутые
- **Повторяющиеся дедлайны** (еженедельно, ежемесячно)
- **Ежедневные напоминания** о ближайших дедлайнах
- **Настраиваемое время** напоминаний
- **Интерактивный календарь** для выбора даты
- **Умные уведомления**: сегодня, завтра, на этой неделе, просроченные

## Использование

### Кнопки
- **Добавить дедлайн** — создать новый
- **Мои дедлайны** — список всех
- **Время напоминания** — настроить уведомления
- **Помощь** — справка

## Технологии

- Python 3.10+
- python-telegram-bot 20.7
- SQLite3
- ConversationHandler
- JobQueue
